# CPE 30天讀書計畫


## 最有效的 CPE 刷題策略

| 步驟            | 做法                                             | 目的           |
| ------------- | ---------------------------------------------- | ------------ |
| **1. 題型優先**   | 先刷高頻題型：Array、Binary Search、DFS/BFS、Graph、DP 基礎 | 覆蓋 70–80% 題目 |
| **2. 每天固定刷題** | 每天 **4 題**（1基礎 + 2中等 + 1偏難）                    | 累積解題經驗       |
| **3. 限時練習**   | 每題 **30–40 分鐘**                                | 培養考試節奏       |
| **4. 建立模板**   | DFS、BFS、Binary Search、Dijkstra、DP              | 減少實作時間       |
| **5. 錯題重做**   | 錯題隔 **2–3 天再做一次**                              | 強化理解         |
| **6. 每週模擬**   | 每週一次 **2–3 小時模擬 CPE**                          | 練習時間分配       |

## CPE 5～6 題最重要的 4 個能力

1️⃣ **DFS / BFS 熟到不用想**  
- 適用於：樹遍歷、圖遍歷、連通分量計算、最短距離計算  
- 建議練習：二元樹 DFS/BFS、圖 DFS/BFS、迷宮走法問題  

2️⃣ **Binary Search 熟練**  
- 適用於：有序陣列查找、邊界問題、數值最優化  
- 建議練習：標準二分搜尋、lower_bound / upper_bound 類題  

3️⃣ **DP（Dynamic Programming）基本題會寫**  
- **Knapsack（背包問題）**：01 背包 / 完全背包  
- **LIS（Longest Increasing Subsequence）**：最長遞增序列  
- **Coin Change（最少硬幣 / 組合數量）**：基本 DP 遞推  
- 建議練習：各種小型 DP 題目，熟悉 state 設計和遞推公式  

4️⃣ **Graph 基本題**  
- **Shortest Path（最短路徑）**：Dijkstra / Bellman-Ford  
- **Connected Components（連通分量）**：DFS/BFS 計算  
- 建議練習：圖遍歷、最短路、聯通性判斷、拓撲排序基礎
