# 台灣四大頂尖大學資訊工程系 C++ 自學手冊

涵蓋學校：
- 台灣大學 (NTU)
- 清華大學 (NTHU)
- 交通大學 (NYCU)
- 成功大學 (NCKU)

---

## 第一階段：程式設計基礎（Programming I / II）
**對應課程**：NTU《計算機程式設計》、NTHU《程式設計一》、NYCU《程式設計入門》、NCKU《計算機概論與程式設計》

### 章節內容
- C++ 環境設定與基本語法
- 資料型別與變數宣告
- 輸入輸出 (`cin`, `cout`)
- 運算子與表達式
- 條件判斷 (`if`, `else`, `switch`)
- 迴圈 (`for`, `while`, `do-while`)
- 函式（定義、呼叫、傳值與傳參考）
- 陣列（1D / 2D）
- 字串（C-style 與 `std::string`）
- 基礎除錯與程式設計邏輯訓練

### 練習題範例
1. 計算一組整數陣列的平均值。
2. 輸入使用者姓名並顯示問候訊息。
3. 判斷一個數是否為質數。
4. 使用函式寫出 Fibonacci 整數序列。
5. 使用 C-style 字串與 `std::string` 比較長度與連接。

---

## 第二階段：進階語言特性與記憶體管理
**對應課程**：NTHU《程式設計二》、NYCU《程式設計實習》、NTU《程式設計進階》

### 章節內容
- 指標與位址操作 (`*`, `&`)
- 指標與陣列關係
- 引用（reference）
- 動態記憶體配置 (`new`, `delete`)
- 結構（struct）與指標應用
- 函式指標與回呼（callback）
- 指標陣列與多層指標
- 記憶體錯誤偵測與除錯觀念

### 練習題範例
1. 寫一個函式，傳入整數陣列，回傳陣列最大值。
2. 使用指標交換兩個整數變數的值。
3. 動態配置一個 N x M 的矩陣並初始化。
4. 使用函式指標實作簡單計算器 (+, -, *, /)。

---

## 第三階段：物件導向程式設計（OOP）
**對應課程**：NTHU《物件導向程式設計》、NCKU《C++程式設計》、NTU《程式設計二》

### 章節內容
- 類別與物件 (`class` / `object`)
- 建構子與解構子
- 成員變數與成員函式
- 封裝（Encapsulation）
- 繼承（Inheritance）
- 多型（Polymorphism）與虛擬函式 (`virtual`)
- 抽象類別與介面設計
- 運算子多載 (Operator Overloading)
- 物件生命週期與拷貝控制（Copy / Move Constructor）

### 練習題範例
1. 設計一個 `Student` 類別，包含姓名、學號與成績，並寫函式計算平均分數。
2. 實作 `Shape` 抽象類別與 `Circle`、`Rectangle` 子類別，計算面積。
3. 使用運算子多載，實作 `Complex` 類別的加法。
4. 模擬銀行帳戶類別，支援存款、提款與查詢餘額。

---

## 第四階段：標準模板庫 STL 與泛型程式設計
**對應課程**：NTU《程式設計實習》、NYCU《資料結構》、NTHU《進階C++應用》

### 章節內容
- 容器 (`vector`, `list`, `map`, `set`, `queue`, `stack`)
- 迭代器（iterator）與範圍 for
- 模板（Template）
- Lambda 表達式
- STL 演算法 (`sort`, `find`, `for_each`, `accumulate` 等)
- 例外處理 (`try`, `catch`, `throw`)
- 智慧指標 (`unique_ptr`, `shared_ptr`)

### 練習題範例
1. 使用 `vector` 儲存整數，找出最大值與最小值。
2. 使用 `map` 統計一段文字中每個單詞出現次數。
3. 使用模板函式寫出兩個數字交換的泛型函式。
4. 使用 Lambda 表達式將 `vector` 中所有數字平方後輸出。

---

## 第五階段：資料結構與演算法實作
**對應課程**：NTU《資料結構》、NTHU《演算法設計》、NYCU《資料結構與實作》

### 章節內容
- Linked List / Stack / Queue
- Binary Tree / BST / AVL Tree
- Heap / Priority Queue
- Graph / BFS / DFS
- Sorting / Searching / Hash Table
- Recursion 與 Divide & Conquer
- 複雜度分析（Big-O, Big-Ω）

### 練習題範例
1. 實作 Stack 與 Queue，並進行操作測試。
2. 用二元搜尋樹 (BST) 儲存整數並做中序遍歷。
3. 實作 Heap 排序。
4. 用 DFS / BFS 搜尋圖形結構中指定節點。
5. 實作 QuickSort 與 MergeSort 並比較效率。

---

## 第六階段：系統與應用實務
**對應課程**：NTU《系統程式設計》、NYCU《作業系統實作》、NCKU《軟體工程》

### 章節內容
- 檔案輸入輸出（File I/O）
- 多檔案專案與標頭管理 (`.h` / `.cpp`)
- 命名空間 (`namespace`)
- C++ 與 C 混合編程
- 錯誤處理與例外機制
- 多執行緒（Thread）與同步機制（Mutex）
- 單元測試與除錯技巧

### 練習題範例
1. 寫程式讀取文字檔，統計行數、單詞數、字元數。
2. 將大型程式拆成多個檔案，並編譯執行。
3. 使用 `try-catch` 捕捉除以零的例外。
4. 使用多執行緒計算矩陣乘法。

---

## 第七階段：綜合專案與高階主題
**對應課程**：NTHU《軟體專案設計》、NTU《系統專題》、NCKU《程式設計實作》

### 章節內容
- 模組化開發與版本控制（Git/GitHub）
- GUI / 網路 / 遊戲開發應用
- C++ 與資料庫連結
- 單元測試（Unit Test Framework）
- 設計模式（Design Patterns）
- 團隊專題（Team Project）

### 練習題範例
1. 實作簡單學生管理系統，包含增刪改查功能。
2. 實作小型網路聊天程式。
3. 使用 GUI 框架（Qt / SFML）做簡單遊戲。
4. 設計並實作一個簡單訂票系統，支援多模組。
5. 使用單元測試框架測試專案功能。

| I | C++ 基礎語法 | 大一上 | 程式邏輯、語法基礎 |
| II | 指標與記憶體 | 大一下 | 底層運作觀念 |
| III | 物件導向程式設計 | 大二上 | 軟體架構與封裝 |
| IV | STL 與模板 | 大二下 | 現代C++技巧 |

| VII | 專題與綜合應用 | 大三下 / 大四 | 團隊實作與創新應用 |
